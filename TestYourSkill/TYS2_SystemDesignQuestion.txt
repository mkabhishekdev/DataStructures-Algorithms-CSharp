Design WhatsApp Chat messaging?

Functionalities:

sendMessage(senderID, receiverID, messageText) — send a message to a user

receiveMessages(userID) — get all unread messages for a user

deleteMessage(messageID) — delete a message by ID

markMessageRead(messageID) — mark message as read

getChatHistory(userID1, userID2) — get all messages exchanged between two users

getActiveChats(userID) — list all users with whom the given user has an active chat
