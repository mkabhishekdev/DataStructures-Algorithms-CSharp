1. 
dotnet clean // to clean the project
dotnet build   //to build project

2. dotnet run // to run project

3. dotnet build -warnas:none // to suppress warnings while building

4. dotnet run -warnas:none // to suppress warnings while running

5. To kill a process or terminate
       - In terminal type: lsof -i :5062
       - Above will show all the processes which are running
       - type the below command to kill the process
             kill -9 <enter the PID here>
